# FastMirror 0.3.6 (2026-03-06)

## 概要
- トレイアイコンクリック時のメインウィンドウ復帰デッドロックを修正しました。
- トレイスレッドと UI スレッド間の ShowWindow 競合を解消し、復帰処理を安定化しました。
- トレイ常駐機能の回帰テストスイートを追加し、今後の変更で動作が壊れないようにしました。

## 変更内容
- トレイスレッドでの ShowWindow/SetWindowPos 直接呼び出しを廃止し、UI スレッド起床後の表示処理へ変更。
- トレイスレッドでは AllowSetForegroundWindow による権限移譲のみを実行し、Condvar notify で UI スレッドを起床。
- UI スレッド側の leave_tray_mode で ShowWindow 等を安全に実行するよう修正。
- hidden ループ脱出時の repaint_after を追加し、eframe イベントループの確実な再起動を保証。
- tests/regression/ にトレイ常駐機能の統合テストを追加（常駐同期・復帰・大ファイル同期）。
- backup.ps1 に統合テスト実行を組み込み、リリース前の自動回帰チェックを実施。
